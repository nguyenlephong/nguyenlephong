## Selected Wins

### 🏢 NDSVN JSC — Senior Software Engineer Lead · *Aug 2025 – Present*

- Took **technical ownership after a CTO transition**, aligning architecture, delivery, rollout systems, and enterprise integrations while leading **8 developers (4 senior) and 3 QC engineers**.
- Engineered a **.NET Core multi-tenant feature-flag platform** for **30+ tenants** and **~40K users** — priority rule engine, percentage rollout (tenant/segment/env), JSON-config values, and stateless runtime evaluation.
- Built secure **Gtel & Napas** integrations — RSA-4096 handshake, AES-256 payloads, **mTLS**, certificate pinning, idempotent contracts, retry queues, and ACK-driven monitoring.
- Shipped **Micro-Frontend** architecture (Angular host + React modules) for independent deploy cycles and a smaller regression surface.
- Stabilized inherited infrastructure across **ArgoCD, CI/CD, domains/SSL, and AKS/FKE/K3S/EKS** clusters — turning recurring unknowns into diagnosable, recoverable systems.
- Built **AI-assisted engineering agents** for code review, deployment support, service-quota checks, API health, and release analysis.
- Rebuilt the **engineering operating model**: org chart, competency matrix, ownership maps, RFCs, runbooks, release tagging, and structured hiring.

### 💬 Zalo PC, VNG — Senior Software Engineer · *May 2024 – Aug 2025*

- Core engineer on **cross-platform Zalo PC** (Web, macOS, Windows) serving **15M+ MAU**; also maintained **Zalo Android** at **~80M users**.
- Delivered **10+ features** — dark mode, download flows, Kiki assistant surfaces, logging, and behavior tracking.
- Built **promotion-flow tracking and monitoring** driving activation across 15M+ MAU.
- Led **Storybook-based design-system** work supporting UI consistency for **50+ developers**.

### 📊 PrimeData VN — Senior Full-stack Engineer · *Jul 2020 – Apr 2024*

- Developed a lightweight **JS SDK under 200KB** with **sub-150ms** load before deeper bundle optimization.
- Established **BDD/E2E coverage** with **100+ scenarios** and **1,000+ test cases**.

### 🚀 Digital SAT Math — Founder & Product Engineer · *2023 – Present (side product)*
